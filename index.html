xTuple ERP

  • Desktop Client: Cross-platform C++ client built using the Qt framework
  • Mobile Web App: Browser-based HTML5 node.js app
  • Database: PostBooks Edition schema based on PostgreSQL

xTuple Web-Enabled Server

  • xTuple Admin Utility: Toolkit for server-side components of the xTuple platform
  • API: Feature-rich REST API modeled off Google's API

xTupleCommerce

  • xDruple: Real-time eCommerce integration with Drupal front end
  • REST Client: Consumes the REST API for all ERP/Drupal integration
