Business Object: IncidentCategory

Definition

Incident categories are used by the incident management system to categorize incidents. TIP: You must manually create incident categories if you want the ability to assign them to incidents. If no incident categories are defined, then none will be available when entering incidents.

Properties:

Name Type Reference
Name String
Description String
Order Number
Email Option Delivery Profile

Related Information

    Other

  • The ID attribute is "name", which will not be automatically uppercased.
  • Used in the crm and billing modules
  • IncidentCategories are lockable.
  • The Incident Category collection is cached.
  • IncidentCategories can be read by anyone but can only be created, updated, or deleted by users with the "MaintainIncidentCategories" privilege.
