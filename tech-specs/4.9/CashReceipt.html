Business Object: CashReceipt

Definition

Cash Receipts may be applied to either an open Invoice or an open miscellaneous Debit Memo

Properties:

Related Information

    Other

  • CashReceipt has no cached defined
  • CashReceipts are lockable.
  • The ID attribute is "number", which will not be automatically uppercased.
  • CashReceipts can be read by any users and can be created, updated, or deleted by users with the "MaintainCashReceipts" privilege.
  • Used in the Billing module
  • Amount applied should be the calculated sum of all applications in the Cash Receipt currency
  • Discount should be the calculated sum of all discounts in the Cash Receipt currency
  • Balance amount is the calculated value of amount - applied
  • 'useCustomerDeposit' should default to the value of setting 'EnableCustomerDeposits'
  • FundsType value should default to 'CHECK'
  • If the funds type is one of the four credit card types then make the following attributes read only: amount, fundsType, documentNumber, documentDate, bankAccount, distributionDate, applicationDate
  • If the cash receipt is posted then the Cash Receipt should be read only
  • When amount or applied values have been updated the balance should be recalculated
  • When the customer is changed on the cash receipt, the currency will be set to that of the customer
  • When the currency or distribution date is changed on the cash receipt, the currency rate for that currency and date should be fetched and set.
  • When one or more cash receipt line items have been added to a cash receipt, customer, bank account and currency will be made read only.
  • If the currency on the Cash receipt does not match the bank account currency a warning dialog should be displayed with message ' This transaction is specified in %1 while the Bank Account is specified in %2.Do you wish to convert at the current Exchange Rate? (%1 and %2 are the Currency abbreviations)
  • An error message should be displayed in the following cases: 1) if the amount to apply is greater than the balance of the cash receipt 2) if the amount to apply plus the discount is greater than the balance of the receivable 3) if a discount is passed where the receivable is a credit
  • Selecting 'Clear' button should clear the applied amount from the cash receipt line
  • Selecting 'Apply Line Balance' button should apply as much of the cash receipt balance as possible to the selected receivable
  • Selecting 'Apply to Balance' button should apply as much cash from the cash receipt as possible to open receivables
