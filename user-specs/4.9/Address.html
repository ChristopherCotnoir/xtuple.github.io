Business Object: Address

Definition

Addresses may be associated with and shared by Contacts, Sites, Ship-to Addresses, Vendors, and Vendor Addresses.

Related Information

    Other

  • The ID attribute is "number", which will not be automatically uppercased.
  • Used in the crm module
  • Addresses are lockable.
  • Address has no cached defined
  • Addresses can be read by users with "ViewAddresses" privilege and can be created, updated, or deleted by users with the "MaintainAddresses" privilege.
  • Selecting 'Edit' button in the 'ADDRESS' panel of a record should open new Address workspace
  • Selecting 'Search' button in the 'ADDRESS' panel of a record should display the Addresses list
  • Editing an address shared by multiple records should display a confirmation dialog 'There are multiple records sharing this address. Would you like to update the address across all of them?'
  • - Selecting 'Yes' in the dialog should update the across all the records
  • - Selecting 'No' in the dialog should update only the current record
  • - Selecting 'Cancel' in the dialog should undo the changes on the current record
