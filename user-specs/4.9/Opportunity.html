Business Object: Opportunity

Definition

Whenever you have a chance to make a sale, it is considered an Opportunity. The xTuple Applications make it easy to track these revenue opportunities with the Opportunity Management system

Properties:

Name Reference
number Displays the system-defined number used to identify the Opportunity
name Enter a name to identify the Opportunity
isActive Select to specify the Opportunity is active. If not selected, the Opportunity will be marked as inactive, causing it to be hidden in some lists
account Specify the CRM Account the Opportunity is associated with
contact Enter the name of the Contact or select a name from the Contacts list. Details for the Contact will be shown after selection
opportunityStage Specify the current Stage of the Opportunity. It's possible an Opportunity will be automatically marked as inactive when it reaches a certain stage, depending on how your system is configured
priority Specify the priority level for the Opportunity
opportunityType Specify the type of Opportunity
opportunitySource Specify the source which generated the Opportunity
amount Enter the monetary value of the Opportunity
currency
probability Project the likelihood the Opportunity will be closed
startDate Specify the date when the Opportunity was first identified. This date will be auto-populated when the Opportunity is created
assignDate Specify the date when the Opportunity was assigned. This date will be auto-populated when the Opportunity is assigned
targetClose Specify the date you expect the Opportunity will close
actualClose Specify the date when the Opportunity closed
notes This is a scrolling text field with word-wrapping for entering Notes related to the Opportunity. Any Notes entered on this screen are for internal purposes only
owner Specify the user who owns the Opportunity
assignedTo Specify the user the Opportunity is assigned to
comments
characterisitcs
documents

Related Information

    Setup

  • Used in the crm module

    Privileges

  • Opportunities can be read by users with "ViewAllOpportunities" privilege and can be created, updated, or deleted by users with the "MaintainAllOpportunities" privilege.
  • Users with "ViewPersonalOpportunities" privilege can read their personal Opportunities but cannot read the Opportunities owned by other users. Users with "MaintainPersonalOpportunities" privilege can create, update or delete their personal Opportunities but not the Opportunities owned by other users

    Settings

  • Opportunities are lockable.
  • When the currency for amount is changed, Amount should be recalculated according to the Exchange rates

    Navigation

  • Contacts search should display only contacts related to the specific account, if account is already selected

    Other

  • The Opportunity has no cache defined
  • Comments panel should exist to add new comments to the Opportunity
  • All Comment Types assigned to Opportunity should be available for selection
  • Documents panel should exist to connect the Opportunities to : Account, contact, Customer, File, Incident, Item, Link, Opportunity, Project, To do
  • TO DO panel should exist to attach existing/new To Dos to the Opportunity
  • User should be able to detach an attached To do from the Opportunity screen
  • User should be able to open an attached To Do from the Opportunity screen
  • Quotes panel should exist to attach existing/new Quotes to the Quote
  • User should be able to detach an attached Quote from the Opportunity
  • User should be able to open an attached Quote from the Opportunity
  • Sales Order panel should exist to attach existing/new Sales Orders to the Opportunity
  • User should be able to detach an attached Sales Order from the Opportunity
  • User should be able to open an attached Sales Order from the Opportunity
